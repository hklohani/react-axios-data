
## Quick Start

```js
import useFetch from 'react-axios-data';

const { data, error } = useFetch({url :'',method: '', body: '', headers: '', params: ''});

*Note: Required Parameter --> url,method
```

## Installation

```console
$ npm i react-axios-data
```
